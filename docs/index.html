Why Use Demeterics?

Multi-Provider Access

Switch between Groq, OpenAI, Anthropic, and Google Gemini with a single credential. No vendor lock-in.

Cost Tracking

See exactly what each workflow costs in real-time. Monitor spend across all AI providers in one dashboard.

Full Observability

Log every prompt, response, and token for debugging and compliance. Understand usage patterns.

LangChain Compatible

Works seamlessly with n8n's AI Agent, LLM Chain, Summarization Chain, and Q&A nodes.

Quick Install

1

Open Community Nodes

Go to Settings > Community Nodes in your n8n instance

2

Install the Package

Click Install and enter n8n-nodes-demeterics

3

Add Your API Key

Get your API key from demeterics.ai and add it to credentials
